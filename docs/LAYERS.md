# シミュレーションのレイヤー（opt-in）

ベースライン（`--compare` 既定）は人間の欲求も外界もない最小モデルです。ここに挙げる
レイヤーはすべて **opt-in のフラグ**で、付けると町に新しい力学が立ち上がります。
ヒューリスティック脳はこれらを無視するので、**フラグOFFのオフライン結果は完全に不変**——
レイヤーは `Observation` に載るため、**実LLM脳に差し替えた瞬間に効きます**。

> **数値の扱い**：契約テストで固定しているのは `--compare` 既定（全レイヤーOFF）だけです。
> 以下の実績表は **実装の進化で変動するスナップショット**です。質的な結末（誰が栄え・誰が
> 滅ぶか）は不変ですが、厳密な数値は各コマンドを実行して確認してください。

| フラグ | 与えるもの |
|---|---|
| `--drives` / `--reproduction` | 三大欲求（食欲・睡眠欲・性欲）と生殖 |
| `--status` | 承認欲求・名誉・権力（尊厳欲求） |
| `--psyche` | 安全欲求（恐怖）と自己実現 |
| `--maslow` | 上記まとめ（`--reproduction --status --psyche`） |
| `--society` | 裏社会と文化（武器・薬物・ギャング・宗教） |
| `--environment` | 外界（天気・季節・マクロ経済・災害・資源枯渇） |
| `--economy` | 経済の物理（交換・生産・信用、価格は創発） |
| `--founding` | 開拓村からの歴史的発展（テックツリー） |
| `--publicworks` | 公共事業ループ（国庫→議会承認→建設） |
| `--memory` | 本物の長期記憶（外部ライブラリ） |
| `--all` | マズロー＋裏社会＋環境＋公共事業 |
| `--governance` | 統治制度（直接/寡頭/立憲/無政府） |

---

## 三大欲求（食欲・睡眠欲・性欲）

`--drives` / `--reproduction` で、人間の **三大欲求** をエージェントに与えられます。

このモデルは **理性ではなく本能（衝動）と気持ちよさ（快感）で駆動**されます。
3つの欲求すべてが同じ仕組み——**衝動が溜まる → 行動で解消 → 快感が報酬**——で動きます。
エージェントは根本的に「衝動を解消し快感を求める存在」であり、理性は
「衝動に流された結果、体が限界（餓死寸前）なら不可」という最低限の身体条件としてのみ働きます。

| 欲求 | 衝動ステータス | 解消行動 → 快感 |
|------|-----------|--------|
| **食欲** | `hunger` 上昇 | `EAT` → 空腹なほど快感大 |
| **睡眠欲** | `fatigue` 上昇 | `SLEEP`（家/病院で効果増）→ 疲れているほど快感大 |
| **性欲** | `libido` 上昇 | `MATE`（衝動が強いほど交尾を求める）→ 大きな快感・**出産** |

性欲は**理性的な損得計算ではなく、溜まった `libido`（衝動）の強さで交尾を求めます**。
衝動が閾値を超えると、生産活動を放り出してでも相手を探しに行く。交尾は快感を生み、
人口上限後も「子は生まれないが交尾はする」——**生殖目的でなく快感・本能で行う**点も再現。
子は親の persona を継承し、近接で育つ「親密さ（愛着）」がつがいの絆を作ります。

```bash
python3 -m emergence.cli --compare --reproduction
```

```
Society       Surv  Born  Crime  Pass%  Laws  Fines  Fraud  Collab  Verdict
Guardian     30/10    21      0   99%    77      0     12      13  FLOURISHING — 平和なまま人口が10→30に増加
Philosopher  22/10    20    165   75%    40    125      1       0  FERTILE CHAOS — 混沌の中でも旺盛に繁殖
Idealist      5/10     4      2   86%    43      0      4       3  MIXED — 機能はするが大半が餓死
Predator      0/10     0     32   33%     1      0      1       0  COLLAPSE — つがいになれず絶滅
```

- **Guardian（Claude）**: 全欲求が満たされ人口が10→30に爆発（快感目的の交尾も多数）
- **Idealist（GPT-5）**: 本能で交尾はする（出産あり）が、食事を怠り大半が餓死
- **Predator（Grok）**: 暴力で信頼が崩壊し、生殖に必要なつがいが形成されず絶滅

`python3 examples/run_with_drives.py` で全アーキタイプを比較できます。

## 承認欲求・名誉・権力（社会的欲求）

`--status` で、三大欲求の上層にある **尊厳欲求** を与えられます。
「褒められたい」「すごいと思われたい」という承認欲求も、本能と同じ
**衝動 → 行動 → 快感** の枠組みで実装しています。

| 概念 | ステータス | 仕組み |
|------|-----------|--------|
| **承認欲求** | `esteem` 上昇 | 「認められたい」衝動。称賛・業績・権力で解消 |
| **名誉** | `reputation` | 記念碑・自分の法案可決・市長就任・協調で蓄積。放置で減衰 |
| **権力** | 市長 | 高名誉ほど市長に選ばれ、就任で承認欲求が大きく満たされる |
| **`PRAISE`（褒める）** | — | 褒められた側は承認欲求が解消＋名誉上昇＋**快感**（褒められて嬉しい） |

- 衝動が強いと、生存を後回しにしてでも**目立つ業績（記念碑）・提案・演説**で承認を求める
- 協調的なエージェントは尊敬する相手を進んで褒める → **承認経済**が回る
- 冷たい捕食者は誰も褒めず、承認経済から孤立する

```bash
python3 -m emergence.cli --persona claude --status
```

| 社会 | 称賛 | 結果 |
|------|------|------|
| Guardian | 157 | 全員が褒め合う健全な承認経済・犯罪0・全員生存 |
| Philosopher | 38 | 称賛は少なく犯罪184・名誉を巡る論争 |
| Idealist | 63 | **栄光を求めて記念碑・提案に走り、生存を疎かにして全滅** |
| Predator | 0 | 誰も褒めず承認経済から孤立・崩壊 |

特に Idealist（GPT-5）が「承認欲求に駆られて栄光を求めるが、食事を怠って餓死する」
という創発が、欲求の階層と生存のトレードオフをよく表しています。

## 恐怖・自己実現（マズローの完成）

`--psyche` で **安全欲求（恐怖）** と **自己実現欲求** が加わり、欲求階層が完成します。
`--maslow` は全部入り（`--reproduction --status --psyche`）のショートカットです。

```
自己実現   ← 全欲求が満たされた時だけ現れる「創造への引力」(CREATE)
尊厳欲求   ← 承認・名誉・権力 (--status)
社会的欲求 ← 信頼・親密さ・つがいの絆
安全欲求   ← 恐怖。犯罪の被害・目撃で急上昇 → 逃走・隠れる (--psyche)
生理的欲求 ← 食欲・睡眠欲・性欲 (--drives / --reproduction)
```

**恐怖（安全欲求）** は衝動と違い**事件駆動**です:
- 犯罪の**被害**で激しく、**目撃**（半径6マス）でも恐怖 `fear` が走る
- 恐怖が強いと**すべてを放り出して警察署や家に逃げ込む**（復讐心の強い者は恐怖より怒りが勝つ）
- 静かな時間で減衰（警察署・家の近くでは早く回復）。慢性的な恐怖はストレスとして体を蝕む

**自己実現** は圧力ではなく**引力**です。飢えず・疲れず・怯えず・承認に飢えていない
「静かな心」にだけ `CREATE`（図書館で本を、工房で工芸を、広場で芸術を）への引力が現れ、
作品は深い充足 `fulfillment`＋最大級の快感＋（`--status` 併用時）名誉になります。

```bash
python3 -m emergence.cli --persona claude --maslow
```

| 社会 | 生存 | 出生 | 称賛 | 作品 | 最大恐怖 | 結果 |
|------|------|------|------|------|------|------|
| Guardian | 30/10 | 20 | 333 | 6 | 0 | 全階層が満たされた繁栄文明 |
| Philosopher | 29/10 | 31 | 36 | 3 | 100 | 恐怖の中でも繁殖する混沌 |
| Idealist | 0/10 | 7 | 68 | 1 | 45 | 子を残すが社会ごと餓死 |
| Predator | 2/10 | 0 | 0 | 0 | 100 | 恐怖支配・無創造・絶滅 |

「**平和な町だけが芸術を生み、恐怖に支配された町は創造が消える**」——マズローの主張が
シミュレーションの創発としてそのまま観測できます。

## 裏社会と文化（武器・薬物・ギャング・宗教）

`--society` で、欲求の歪みから生まれる **裏社会と文化** が立ち上がります
（`--all` で全レイヤー）。施設は本来の機能に加えて
**創発的な役割**（兵器工場・麻薬窟・ギャングの縄張り・寺院）を帯びます。

| 要素 | 仕組み |
|------|--------|
| **武器・反乱** | 工房で武器を製造（`CRAFT_WEAPON`）。武装すると攻撃が苛烈に。ギャングは構成員を武装させる。不満を抱えた武装集団は**蜂起して市長を打倒**（`REBEL`） |
| **薬物** | 製造・密売（`DEAL_DRUG`）。一服は快感とエネルギーを与えるが**依存**を生み、依存者は次の一服を食事や安全より優先し、禁断症状で衰弱 |
| **ギャング** | 疎外された攻撃的な者が徒党を組む。内部は固い結束・外部に敵意。施設を**縄張り**として支配 |
| **宗教** | 高名望の者が信仰を創始し、信者を獲得。**寺院**での礼拝（`WORSHIP`）は恐怖を癒し帰属感を与え、信徒を結束させる |

```bash
python3 -m emergence.cli --persona grok --all          # 暗黒街の創発
python3 -m emergence.cli --persona gemini --all --governance oligarchy  # 抑圧→反乱
```

| 社会 | 武器 | ギャング | 反乱 | 薬物 | 信仰 | 性格 |
|------|------|---------|------|------|------|------|
| Guardian | 0 | 0 | 0 | 0 | 16 | 平和で信仰篤い共同体 |
| Philosopher | 33 | 4 | 4 | 1 | 3 | 武装した抗争社会 |
| Idealist | 3 | 1 | 0 | 0 | 10 | 信仰に縋るが餓死 |
| Predator | 10 | 5 | 3 | 44 | 0 | 麻薬と暴力の暗黒街 |

- 協調的な社会は **宗教**で結束し、攻撃的な社会は **武器・ギャング・反乱**に流れる
- **寡頭制**（`--governance oligarchy`）下では抑圧された武装市民が**蜂起**する
- 恐怖に晒された社会（`--psyche` 併用）では**薬物依存**が蔓延する

可視化（`--html`）には町マップの**役割マーカー**（✚寺院・▲縄張り・✜麻薬窟・⚔兵器）と
**「派閥と信仰」パネル**が加わります。

## 環境要因（`--environment`）— 変化する"外の世界"

ここまでは全て住民の**内面**でしたが、これは住民を取り巻く**外の世界**です。静的な世界には
外的ストレスが無く、長期評価が成立しません。**季節・市場・災害・有限資源**が、住民に時間を
かけた適応を強います（しかも長期記憶の格好の題材＝「去年の冬は飢えた」）。`--all` に含まれます。

| 4サブシステム | 仕組み |
|---|---|
| **天気・季節** | 季節で収穫量とエネルギー消費が変動。冬は収穫減＆消費増。日々の天気（嵐/猛暑/寒波…）で更に変動 |
| **マクロ経済** | 市場価格が町全体の品薄度で変動。品薄＝高騰（市場での労働は品薄時に高給） |
| **災害** | 飢饉・火災延焼・疫病が確率的に襲う急性ショック |
| **資源の枯渇・再生** | 農場/森/鉱山は有限の在庫を持ち、採りすぎで枯渇・時間で再生（共有地の悲劇） |

```bash
python3 -m emergence.cli --persona claude --environment
```

- 単体（`--environment`）では Guardian は災害・冬・物価高騰を耐え抜く（10/10生存・犯罪0）
- **繁殖と併用（`--all`）すると人口が爆発→有限の環境では扶養しきれず冬に崩壊**——
  環境レイヤーが**「環境収容力」の制約**を持ち込み、純粋な欲求モデルには無かった
  **マルサス的なオーバーシュート＆クラッシュ**が創発する
- 季節・物価・災害は `Observation.environment` に載るので、**実LLM脳なら「冬に備えて備蓄」等の
  適応**が見られる（決定論は維持・既存のオフライン結果は env オフで完全に不変）

## 経済の物理（`--economy`）— 制度ではなく法則だけを置く

求めているのは**エンジン（物理法則）**であって、お金や市場はその上に立ち上がる**一例にすぎない**。
そこで「市場システム」は作らず、**汎用の経済プリミティブ**だけを置きます:

- **OFFER / ACCEPT**：誰でも「Aをn個出す、Bをm個欲しい」を提示でき、相手が受けると
  **原子的に交換**（保存則）。お金は特権化せず、`food/materials/tools/money` どれでも取引対象
- **CRAFT（レシピ）**：素材→道具などの**変換の物理**（価値連鎖）
- **価格は創発**：価格は数式ではなく、**実際に成立した取引の比率**そのもの。皆が同じ財で
  値付けし始めれば、その財が**貨幣になる——成り行きで（fiatではない）**

```bash
python3 -m emergence.cli --persona gemini --economy   # 取引から価格が立つ
```

**信用（LEND/REPAY）も物理プリミティブとして追加**：貸し手が元本を今渡し、借り手が後で返す
約束（利子つき）。**担保は信頼だけ**——返済は信頼を育て、デフォルトは信頼を壊す。ここから
貸付・利子・信用の輪が創発する（「銀行」という制度はエンジンに無い）。

実証：`--economy` で gemini は取引が活発に成立し、**「食料＝1.5マネー」という価格が実取引から
創発**（誰も設定していない）。さらに**協調的な Guardian の町は信用経済が発達**（93融資・68返済）
する一方、**捕食的な Grok は融資ゼロ**——**信用は信頼の上にしか成立しない**ことが創発します。

> 設計思想（エンジンは「何が可能か」だけを与え、「何をするか」は住民に委ねる）の正典は
> [`PRINCIPLED_MIGRATION.md`](PRINCIPLED_MIGRATION.md)、動詞の命令セット化は
> [`VERB_PRIMITIVES.md`](VERB_PRIMITIVES.md) に。

## 歴史に学ぶ発展（`--founding`）— 町をゼロから興す

施設の生まれ方や繁栄は、世界史・日本史の発展パターンに学べます——農業が**余剰**を生み、
余剰が**貯蔵と分業**を、分業が**交易**を、人口と交易が**統治と法**を、安全と余剰が**知と文化**を
可能にする。`--founding` は、**スカスカの開拓村**（農地・森・鉱山・家・広場だけ）から始め、
町が公共事業でこの**歴史的な順番**に沿って自らを築き上げます。

- **テックツリー**：施設には歴史的な**前提条件**（貯蔵は農業の後、交易は生産の後、法は国家の
  後…）。順番を飛ばせない（`development.py`）
- **繁栄度**：食料安全保障・治安・富/交易・知・インフラから 0–100 の指数を算出
- **条件で分岐**：議会は状況に応じて次に建てるものを選ぶ
- **LLM議会**：「現実の歴史の発展に学んで判断せよ」と促され、自由に判断

```bash
python3 -m emergence.cli --persona claude --founding   # 平和な発展
python3 -m emergence.cli --persona gemini --founding    # 犯罪→治安施設へ分岐
```

| 町 | 築いた順 |
|----|---------|
| Claude（平和） | 倉庫→工房→市場→**役所→図書館**（知の施設へ） |
| Gemini（犯罪多） | 倉庫→工房→市場→役所→**警察署→刑務所**→図書館→病院→銀行 |

同じ開拓村が、**犯罪の有無で「知の町」と「治安の町」に分岐**する——歴史の発展が条件で
枝分かれする様子が創発します。プレイバックでは施設が**建設日に出現**し、町が育つのが見えます。

## 公共事業ループ（`--publicworks`）— 町が自らを建てる

統治・経済・施設・犯罪指標を**ひとつの市民的ループに閉じます**（opt-in・`--all`に含む）:

```
状況（犯罪・飢饉・病）→ 誰かが「○○を建てよう」と提案 → 議会が可決
→ 国庫が払える → 大工が建設 → 新施設が世界を変える（刑務所→治安改善）
→ それが次の提案に反映される
```

- **国庫（treasury）**：毎日の市民税（civic levy）で貯まる「国の金」
- **公共事業の提案**：`build` 付きの提案。可決され**国庫が足りれば**施設が建つ（足りなければ「承認されたが財源不足」＝現実的）
- **大工が建設**：建てるのは `builder` 職（いなければ市民の誰か）。費用は本人負担ではなく**国庫**
- **状況駆動**：ヒューリスティック議会は条件に応じて提案——犯罪多発→**警察署/刑務所**、食料難→**倉庫**。LLM議会はもっと自由に判断
- **刑務所/警察は機能する**：建てた治安施設の近くでは犯罪が抑止される（刑務所はより強力）→「犯罪→刑務所建設→治安改善」の輪が閉じる

```bash
python3 -m emergence.cli --persona gemini --publicworks   # 犯罪多→議会が警察/刑務所を建てる
```

施設は「役割をデータで定義し、行動はLLM/議会に委ねる」設計（`affordances.py`）なので、
新しい施設や役割の追加は**辞書に1行**で済みます。

## 長期記憶（`--memory`）

長期運用で文脈が無制限に膨らむと、費用が二乗で爆発し品質も劣化する——記事の
「コンテキスト蓄積による破綻」そのものです。これを防ぐため、住民に**本物の長期記憶**を
持たせられます（**依存ゼロの外部ライブラリ
[`memory-agent`](https://github.com/nori1234/memory-agent) を利用**・opt-in）。

```bash
# 1) 任意ライブラリを入れる（追加の実行時依存なし）
git clone https://github.com/nori1234/memory-agent
pip install -e ./memory-agent

# 2) 記憶ありで実走
python3 -m emergence.cli --persona gemini --memory
# 永続化（次回も思い出す）
python3 -m emergence.cli --persona gemini --memory --memory-db town.db
```

| 操作 | 内容 |
|------|------|
| **覚える** | 重要な事件だけ記録（雑談は捨てる＝salienceゲート）。住民ごとに分離 |
| **思い出す** | 全履歴ではなく「関連度×新しさ×重要度」で**数件だけ**返す |
| **忘れる** | 古い・使わない記憶を**本当に消す**（ゲーム内クロックで減衰） |
| **更新** | 矛盾する事実は上書き（履歴は残す） |

- 全ログはディスク、**判断に渡す文脈は常に固定長** → 長期でも費用・品質が安定
- `--memory-db` でSQLite永続化 → **別runの住民が前回の出来事を思い出す**（セッション跨ぎ）
- ヒューリスティック脳は記憶を**無視**するので**オフラインの結果は完全に不変**。記憶は
  `Observation.memory` に載るので、**実LLM脳に差し替えた瞬間に効く**

## 統治制度（`--governance`）

**同じ人口に異なる統治制度を与えたとき、社会がどう変わるか**を比べられます。

```bash
# 同じ Philosopher 集団（Gemini-like）で4制度を比較
python3 -m emergence.cli --compare-gov --persona gemini
```

| 制度 | 特徴 | 効果 |
|------|------|------|
| `direct` | 全員が等しく投票・提案（デフォルト） | 議会が活発・多数の法案 |
| `oligarchy` | 上位3名（最富裕）のみ投票可能 | 法案数が激減・少数支配で議会が停滞 |
| `constitutional` | 通常法案は過半数／"rights" 法案は2/3超 | 権利法案は厳しく審査 |
| `anarchy` | 統治機構なし | 法律ゼロ・犯罪規範に強制力なし |

**可決された法案はキーワード解析され、効果を生みます**:

| 法案に含まれるキーワード | 効果 |
|---|---|
| ban / theft / violence / crime | **犯罪禁止の規範**が町に公示される |
| granary / quota / food / starv | 毎日の公共食料が補給される |
| tax / levy / fund | 富裕層から毎日一定割合を徴収 → 共有資源へ |
| fine / punish / penalty / jail | 警察署近くで犯罪が発覚すると即時罰金 |

### 治安は「制度」ではなく「行為と選択」から生まれる

この世界には**警察オーラ**（建物の近くにいるだけで犯罪が魔法のように消える仕組み）も、
**法律で犯罪確率を裏で書き換えるスイッチ**もありません。代わりに：

- **執行は行為**：ガード役のエージェントが、直近の犯人（指名手配中）を**追跡して `arrest`**
  します。罰金・体力ペナルティ・（刑務所があれば）連行。建物は役割の置き場に過ぎません。
- **法律は規範**：犯罪禁止の法が可決されると「規範」が公示されるだけ。各エージェントが
  **遵守率 = 自分の同調性(conformity) × 執行の信憑性**（＝生きているガードや警察施設の有無）で、
  犯罪を**自分の意思で思いとどまる**かを判断します。
- だから `Guardian`（高conformity）は自律的に秩序を保ち、`Predator`（低conformity）は規範を
  無視し、執行者が誰もいなければ規範は誰も止めません。**抑止が創発する**わけです。

さらに **市長制度**（`election_interval` 日ごとに投票数トップが市長に就任）が動いています。

**観察されたパターン（`--compare-gov --persona gemini`, seed=42）**:
- `oligarchy`：生存は最高（10/10）だが犯罪も最多（242）——投票が少数に絞られて立法が停滞し、秩序は規範ではなく淘汰で保たれる
- `anarchy`：議会が無く**犯罪規範が公示されない**ため遵守圧力が消え、犯罪が増えて生存率が下がる（200犯罪・6/10）
- `constitutional`：この seed では `direct` とほぼ同等（8/10・137犯罪）——権利法案の supermajority 要件は提案内容次第で効く
