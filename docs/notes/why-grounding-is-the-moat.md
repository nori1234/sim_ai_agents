# なぜ接地が「堀」であり「最前線」なのか — 外部フレームのメモ

> 運用は HANDOFF、技術は GROUNDING.md。これは事業的・思想的な *why* の文脈メモ
> （2つの外部フレームを我々の数値に接続する）。判断の根拠であって、判定は G2 が持つ。

## 1. フィジカルAI の「4層」地図 — 本当の堀は第4層ループの“中身”

産業側の地図（WFM / デジタルツイン / 産業OS / 現場運用）は方向として正しい：
2030年以降の競争力は「現場を長く深く AI に学習させ続けたか」に移る、という主張。

- **WFM（世界基盤モデル）** ＝ 物理法則の共通言語（NVIDIA Cosmos 系）。全AIが立つ「大地」。
- **デジタルツイン** ＝ 仮想での無リスク試行。
- **産業OS** ＝ 制御権・規格（シーメンス等が握りにいく神経系）。
- **現場運用ループ** ＝ 「使うほど賢くなる」循環。★ここが moat と言われる。

**しかし地図が一言で済ませる第4層ループは、自動では閉じない。** 現行モデルの既定動作は
**訓練の再生（replay）**——現場で不可逆な帰結を浴びても、その一生の中で信念を更新して
行動を変える機構が無い。方策が反射なら、データを積んでも「反射のまま量が増える」だけ
（データ死蔵）。∴ **本当の堀は「蓄積した現場データ」ではなく「それを接地に変換できる
学習機構」**。機構が無ければ現場独占は堀にならない。我々の数値がこれを裏づける：
**G1（生の regime 差）は反射で通過可能／記憶+credit 族は G1≈+0.09 で頭打ち／真の接地
G2（同一状況での挙動差）は記憶なし方策では原理的に ≤0**。

日本への含意：身体が標準化され WFM を外部が供給する世界で、伝統的強み「現場のカイゼン＝
暗黙知を回して賢くなる循環」を *AI の中で* 再現する機構を持たない側は、現場を持っていても
堀にできない。逆に **接地機構（lived consequence → 行動更新）を持てば、現場データの蓄積が
初めて複利で効く**。

## 2. アナログ／連続性の系譜（Dyson 的直観）— 接地は「程度」でなく「種類」

3つの直観と我々の数値は、同じ壁を別方向から指す：

- **連続体仮説（可算 vs 非可算、間に中間なし）** → 接地は程度でなく **質的断絶**。整数を
  いくら数えても連続体に届かないように、**反射をいくらスケールしても接地に届かない**。
  「族が G1≈+0.09 で頭打ち」はこの越えられないギャップの数値版。データ量・モデルサイズ
  という可算軸の延長に接地は無い。
- **意味は論理で定義できず、関係のマッピングから立ち上がる** → 接地も論理規則では証明
  できない。反射（`money≥12→預金` の離散 if-else）は G1 を通すが接地ではない。真の接地は
  **同一状況での連続的な挙動差＝関係性（G2）** でしか測れない。「意味＝関係」と「接地＝
  帰結と行動の関係」は同型。
- **機械が連続性を獲得し、デジタル→アナログへ** → 我々の north star の別名。**接地＝機械が
  不可逆な現実と“連続性(continuity)”を持つこと**。(3)推論 + (4)行動化 は、離散的な訓練再生
  から、世界と地続きの連続適応へ渡る一段。

## 3. だから我々がやっていること

sandbox はこの4層のミニチュア：engine の経済則（隠れ demurrage regime＝不可逆な帰結）＝
WFM/物理、sandbox＝ツイン、方策＝産業OS/制御、そして **B2（帰結→信念→行動）・C1（反射
では解けない課題）・D1（(3)推論と(4)行動化の分離）** ＝ 第4層ループが**そもそも閉じるか**の
最小実証。我々は地図の消費者ではなく、地図が「ここが堀」と指す空白（離散→連続、replay→
接地の質的断絶を機械で越えられるか）の feasibility を測っている。それが事業的な堀であり、
同時に技術の最前線である理由。

判定は常に G2（反射で偽装できない）に置く。この文脈は *なぜ* を支えるが、*できたか* は
測定が答える。
